뮤지컬 수업 스튜디오 – README
=================================
(Last updated: 2025-08-15 02:40 UTC)

◆ 개요
- 단일 HTML로 동작하는 '뮤지컬 수업 스튜디오' 사용 안내입니다.
- 자동저장(LocalStorage), 내보내기(JSON/TXT/PDF), 공유 링크, 라이트/다크, KOR/ENG, 모바일 하단 퀵네비 지원.

──────────────────────────────────
1) 빠른 시작
──────────────────────────────────
1. HTML 파일을 Chrome/Edge/Safari로 엽니다.
2. 상단에서 프로젝트 정보를 입력합니다.
3. 좌측(PC) 또는 하단(모바일) ‘진행 단계’ 버튼으로 단계별 작성 화면으로 이동합니다.
4. ‘결과 미리보기’에서 전체 내용을 확인하고 ‘내보내기’로 저장하세요.

──────────────────────────────────
2) 내보내기 & 파일 열기(복원)
──────────────────────────────────
- 내보내기: JSON(백업/복원용), TXT(텍스트), PDF(A4 비율·한글 지원).
- 파일 열기: 이전에 저장한 JSON을 선택하면 상태가 복원됩니다.

※ PDF 팁
- ‘미리보기 최신화’ 후 내보내기를 권장합니다.
- 브라우저 확대/축소 100%를 권장합니다.

──────────────────────────────────
3) 공유 링크
──────────────────────────────────
- ‘공유 링크 복사’ 클릭 시 현재 상태가 URL 해시(#data=…)로 인코딩되어 복원 가능한 링크가 복사됩니다.
- 민감 정보가 포함될 수 있으니 공유에 주의하세요.

──────────────────────────────────
4) 활동 구성/브랜딩
──────────────────────────────────
- [과정 5] ‘준비하기/떠올리기/표현하기/나누기(공유)’ 4열. ‘활동 추가’로 세로 카드가 아래로 생성됩니다.
- [과정 6] 캔버스(16:9, 1920×1080): 브러시 색·굵기(1–80), 펜/지우개, 캔버스 지우기, 이미지 업로드. 별도 저장 없이 자동 포함.

──────────────────────────────────
5) 기타
──────────────────────────────────
- 자동저장: 브라우저 LocalStorage 사용.
- 초기화: 상단 ‘초기화’ 클릭 → 확인 시 모든 데이터 삭제.
- 테마/언어: 🌙/☀️, KOR/ENG 전환 지원.
- 모바일: 상단 버튼 아이콘 UI + 하단 퀵네비(내용 출처는 팝업으로 표시).

제작자: 교육뮤지컬 꿈꾸는 치수쌤 (https://litt.ly/chichiboo)
